    • Developer Cog

    The Developer cog provides essential commands for bot developers, including the ability to manage cogs by loading, reloading, and unloading them. The cog also allows for the execution of SQL queries and Python code with its 'sql' and 'eval' commands.

    Additionally, the 'blacklist' command can be used to block users or channels from accessing the bot, and the 'dm' command allows for the bot to send direct messages to users. Lastly, the cog provides a 'guildfetch' command for retrieving information on a specific server.

    If a command is suffixed with "- G", it indicates that it is a group command which comprises of two or more sub-commands.

    "<>" symbols mean the particular argument is required | "[]" symbols mean the particular argument is optional.You don't have to type out the symbols when invoking the command.

    blacklist - G

    Group of commands to black objects from using Geralt.

    .gblacklist remove

    snowflake

    Can be used 2 times every 5.0 seconds per user.

    Nil

    Can be used 2 times every 5.0 seconds per user.
