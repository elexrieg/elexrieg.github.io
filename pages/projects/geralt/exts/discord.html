    • Discord Cog

    The Discord cog provides various utility commands to help with managing and exploring Discord servers. From quickly checking a user's avatar or Spotify activity to viewing server and user information, this cog has got you covered. Additionally, it includes snipe command to keep track of deleted and edited messages, making it an essential tool for moderation.

    If a command is suffixed with "- G", it indicates that it is a group command which comprises of two or more sub-commands.

    "<>" symbols mean the particular argument is required | "[]" symbols mean the particular argument is optional.You don't have to type out the symbols when invoking the command.

    avatar

    .gavatar

    user=None

    av | dp | pfp | pp

    See the user's PFP in an enlarged manner.

    banner

    .gbanner

    user=None

    Nil

    See the user's banner in an enlarged manner.

    ping

    .gping

    pong

    Get proper latency timings of the bot.

    serverinfo

    .gserverinfo

    gi | si

    Get entire details about the guild.

    snipe - G

    .gsnipe

    s

    Get edited / deleted messages.

    Can be used 3 times every 5.0 seconds per user.

    .gsnipe delete

    index

    channel

    user

    d | del

    Sends details on recently deleted messages.

    Can be used 3 times every 5.0 seconds per user.

    .gsnipe edit

    channel

    user

    index=0

    e | ed

    Get the details of the recently edited messages.

    Can be used 3 times every 5.0 seconds per user.

    .gsnipe stats

    flag

    Nil

    Get guild and global snipe stats.

    --globalstats Sends global stats for if arg set to True, else sends guild's.

    .gsnipe stats [--globalstats True/False]

    spotify

    .gspotify

    user=None

    sp | spot

    Get information on what a user is listening on spotify.

    userinfo

    .guserinfo

    user=None

    ui | user

    Get entire details about a user.
