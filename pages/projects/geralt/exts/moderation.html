    • Moderation Cog

    The moderation cog provides essential tools for managing and moderating your Discord server. From banning and kicking users to muting and changing nicknames, this cog offers a suite of powerful features to help maintain order and discipline in your community.

    With the ability to lock channels and remove unwanted users, the moderation cog is an essential tool for any server administrator.

    If a command is suffixed with "- G", it indicates that it is a group command which comprises of two or more sub-commands.

    "<>" symbols mean the particular argument is required | "[]" symbols mean the particular argument is optional.You don't have to type out the symbols when invoking the command.

    ban

    .gban

    user

    reason=Not Provided

    Nil

    Teach them a lesson by banning them out.
