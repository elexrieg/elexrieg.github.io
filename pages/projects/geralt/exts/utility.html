    • Utility Cog

    The Utility cog is a collection of handy commands that can make your Discord experience more efficient and productive. With features like AFK, Avatar History, and User History, you can keep track of important information and stay on top of your game. The Highlight command sends you a DM when someone mentions specific words you set, so you never miss an important message. And with the Todo command, you can create and manage your to-do list without ever leaving Discord.

    Finally, the User Log command gives you the ability to track changes made to your username, avatar, and discriminator, providing you with a complete picture of your activity on Discord.

    "<>" symbols mean the particular argument is required | "[]" symbols mean the particular argument is optional.You don't have to type out the symbols when invoking the command.

    highlight - G

    .ghighlight

    hl

    Get notified by triggered words!

    .ghighlight add

    trigger=None

    set

    Add trigger words to notify you.

    .ghighlight remove

    trigger=None

    del

    Remove trigger words to notify you.

    .ghighlight list

    all

    See all of your trigger words.

    .ghighlight block

    object

    bl | blacklist | lock

    Block users or roles from highlighting you!

    .ghighlight unblock

    object

    unlock | whitelist | whl

    Unblock users or roles from highlight you!

    .ghighlight blacklisted

    blcklist | blocked | blocked-list

    Returns List of objetcs you've blocked.

    todo - G

    .gtodo

    td

    Sends Todo sub - commands.

    Can be used 2 times every 5.0 seconds per user.

    .gtodo add

    task

    Nil

    Add tasks to your todo list.

    Can be used 2 times every 5.0 seconds per user.

    .gtodo remove

    task_id

    done | finished

    Remove a particular task.

    Can be used 2 times every 5.0 seconds per user.

    .gtodo edit

    task_id

    edited

    Nil

    Edit a particular task.

    Can be used 2 times every 5.0 seconds per user.

    .gtodo list

    all

    See your entire todo list.

    Can be used 2 times every 5.0 seconds per user.

    .gtodo clear

    cl | del | delete

    Delete your entire todo list.

    .gtodo show

    task_id

    see

    Check out a task in detail.

    Can be used 2 times every 5.0 seconds per user.
